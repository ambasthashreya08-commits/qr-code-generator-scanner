### ⚡ Generate • Scan • Decode QR Codes Instantly

A modern QR Code Generator & Scanner built with **Python, Streamlit, OpenCV, NumPy, and Pillow**.

</div>

---

## 🎯 Project Overview

This application enables users to:

✅ Generate QR codes from URLs or text

✅ Download generated QR images

✅ Upload QR images for scanning

✅ Decode QR content instantly

✅ Use a simple and interactive web interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ambasthashreya08-commits/qr-code-generator-scanner.git
cd qr-code-generator-scanner
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
